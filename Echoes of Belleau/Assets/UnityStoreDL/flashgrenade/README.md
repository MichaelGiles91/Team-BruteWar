--- Flash Grenade ---



Made by MiyopGames.



This pack includes a flash grenade game-asset, ready to use.



The model and it's textures are intended for first person shooters. Polycount is similar to current generation games.

Materials, meshes and texture names follow an easy to understand naming convention and each weapon material has the same name as the textures for it.



The model has 4K textures. All values are with attachments included. Created using Blender and Substance Painter.



* Assets are accompanied with the standard 4 textures (base, mask, normal and emissive) and all textures are 4096x4096.
* The models are created with performance in mind and are industry standard poly counts and appropriate amount of materials.
* This model uses PBR (Physically-Based Rendering) workflow.
* Works both on URP and HDRP. Separate packages for either render pipelines.
* Includes normal, emissive and AO maps.
* No type of AI tool has been used in creation of this asset.
* Fully UV mapped.
* Files format:

\- FBX for models and animations

\- PNG for textures
